# 📓 Note‑Taking App — 3 languages

A simple CLI note‑taking tool that allows you to **add**, **list**, and **delete** notes. Notes are stored in a local JSON file. Implemented in **Python**, **JavaScript (Node.js)**, and **Go**.

## 🚀 Run

| Language | File | Run command |
|:---|:---|:---|
| Python | `python/notes.py` | `python notes.py` |
| JavaScript | `javascript/notes.js` | `node notes.js` |
| Go | `go/notes.go` | `go run notes.go` |
